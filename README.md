# 違反勞基法之資料探勘分析

本專案利用政府公開之勞動法規行政裁罰資料，
透過關聯規則與序列樣式探勘方法，
分析事業單位之重複違規行為與時間結構特性。

---

## 專案目標

1. **辨識常見違規組合**  
   從大量行政裁罰紀錄中，找出重複出現之違規樣態。

2. **分析違規順序特徵**  
   檢視違規是否具有固定順序或重複流程。

3. **探討違規時間結構**  
   分析連續違規之時間間隔是否存在關聯性。

---

## 方法架構

### 關聯規則探勘（Association Rule Mining）

- 以單筆裁罰案件為交易單位  
- 建立頻繁項目集與規則模型  
- 分析違規法規之共同出現關係  

---

### 序列樣式探勘（Sequential Pattern Mining – PrefixSpan）

- 以同一事業單位為分析單位  
- 依違規時間排序建立違規序列  
- 探討違規行為是否存在重複順序模式  

---

### 連續違規時間間隔分析

- 依 PrefixSpan 的結果挑選「具代表性的重複違規序列」（例如同一法規於三個不同時間點重複出現之序列）作為延伸分析對象 
- 計算：
  - 第 1–2 次違規間隔  
  - 第 2–3 次違規間隔  
- 進行統計檢定，檢視兩段時間間隔是否具關聯性  

---

## 主要發現

- 關聯規則分析顯示，違規法規之出現並非隨機分布，而存在可重複出現之違規組合樣態。  
- 序列樣式探勘發現，部分事業單位之違規行為具有明確時間順序特徵，特定違規後常伴隨再次違規或延伸至其他相關法規。  
- 在序列樣式探勘結果中，觀察到「同一法規於不同時間點重複出現三次」的序列樣態；針對此類案例進行延伸分析發現，第 1–2 次與第 2–3 次違規的時間間隔存在關聯性，顯示重複違規具有行為延續特徵。 
- 本專案驗證資料探勘方法可有效從行政裁罰資料中萃取具解釋力之行為樣態，具政策分析與風險監測潛力。
