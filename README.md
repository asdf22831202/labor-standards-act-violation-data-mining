# 違反勞基法之資料探勘分析
## 專案目標
本專案以台灣政府公開之勞動法規行政裁罰資料為基礎，透過資料探勘與統計分析方法，探討事業單位在違反勞動法規時，是否存在重複且具有結構性的違規行為模式。

此專案的核心目標在於:
- 從大量行政裁罰紀錄中，找出常見的違規組合與行為樣態
- 分析違規是否具有「先後順序」與「重複發生」的特性
- 將複雜的分析結果轉化為可解讀的結論與觀察

## 分析方法概述
本專案結合多種資料探勘技術，從不同角度分析違規行為：
- **關聯規則探勘（ Association Rule ）**
  
用以找出經常同時出現的違規法規組合，協助理解那些違規行為傾向伴隨發生。
- **序列樣式驗勘（Sequential Pattern Mining, PrefixSpan）**
  
分析同一事業單位的違規順序，探索是否存在固定的重複違規流程。
- **時間間隔與統計分析**
  
計算重複違規之時間間隔，並透過統計檢定檢視時間間隔之間是否具相關性。

## 分析成果
透過本研究的分析流程，得到下觀察：
- 在未考慮時間順序的關聯規則分析中，結果顯示事業單位或負責人在違反勞動法規時，其所涉及的法規並非隨機分布，而是存在可重複出現的違規組合樣態，顯示不同法規之間可能具有關聯性的違規行為模式。
- 進一步納入時間順序進行序列樣式分析後發現，部分事業單位的違規行為具有明確的重複順序特徵，例如在違反特定法規後，後續再次違反相同法規，或接續違反其他相關法規，且此類違規序列在資料中多次出現。
- 針對連續違反同一法規三次以上之個案進行延伸分析後發現，第 1 次與第 2 次違規之時間間隔，與第 2 次與第 3 次違規之時間間隔之間存在關聯性，顯示重複違規在時間上並非獨立事件，而具有一定的行為延續特性。
- 整體而言，本專案顯示資料探勘方法能有效從大量行政裁罰資料中，萃取具解釋價值的違規行為樣態，並具備實務分析上的應用潛力。
